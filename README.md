# Raspberry Pi Sense HAT

>Sense HAT je přídavná senzorová karta pro Raspberry Pi, vyrobená speciálně po misi Astro Pi.

Obsahuje matici 8x8 RGB LED, pětitlačítkový joystick a následující senzory:
* Gyroskop
* Akcelerometr
* Magnetometr
* Teploměr
* Tlakoměr
* Vlhkoměr
